# Privacy Policy

Adanos Sentiment Lens is designed to keep user data minimal and local.

## Data Stored Locally

The extension stores the following data in Chrome local storage:

- Adanos API key
- Default sentiment source
- Default analysis period in days
- Stock watchlist
- Last selected context-menu ticker

## Data Sent to Adanos

When the user requests a sentiment lookup, the extension sends:

- The selected stock ticker symbols
- The selected analysis period
- The selected stock sentiment source
- The user's Adanos API key in the `X-API-Key` header

Requests are sent only to `https://api.adanos.org`.

## Data Not Collected

The extension does not collect or transmit:

- Browsing history
- Page contents
- Personal financial account data
- Payment data
- Analytics events
- Advertising identifiers

## Limited Use

Data is used only to provide the extension's single purpose: stock market
sentiment lookup for user-selected tickers.

The use of information received from Google APIs will adhere to the Chrome Web
Store User Data Policy, including the Limited Use requirements.

## Contact

For privacy questions, contact: contact@adanos.org
